# JobShield Dataset

## Overview

This folder contains the dataset used for the **JobShield** project, which aims to detect fraudulent online job advertisements using Machine Learning and Natural Language Processing (NLP).

The dataset serves as the primary source for data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and evaluation.

---

## Dataset Information

- **Dataset Name:** Fake Job Postings Dataset
- **Project:** JobShield – Detecting Fraudulent Online Job Advertisements Using Machine Learning
- **Task:** Binary Classification
- **Target Variable:** `fraudulent`
  - `0` → Genuine Job Posting
  - `1` → Fraudulent Job Posting

---

## Dataset Features

The dataset contains various job-related attributes, including:

- Job Title
- Location
- Department
- Salary Range
- Company Profile
- Job Description
- Requirements
- Benefits
- Employment Type
- Required Experience
- Required Education
- Industry
- Function
- Telecommuting
- Company Logo Availability
- Screening Questions
- Fraudulent (Target Variable)

---

## Purpose

This dataset is used to:

- Perform Exploratory Data Analysis (EDA)
- Handle missing values and preprocess data
- Convert textual data into numerical features using TF-IDF
- Train and compare multiple Machine Learning models
- Evaluate classification performance
- Build a fake job posting detection system

---

## Machine Learning Algorithms

The following algorithms will be evaluated during this project:

- Logistic Regression
- Support Vector Machine (SVM)
- Naïve Bayes
- Decision Tree
- Random Forest
