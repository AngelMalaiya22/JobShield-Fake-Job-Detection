# JobShield-ML

## Project Overview

JobShield is a Machine Learning project that detects fraudulent online job advertisements using Natural Language Processing (NLP) and classification algorithms.

## Objectives

- Detect fake job postings
- Compare multiple ML algorithms
- Perform data preprocessing and feature engineering
- Evaluate model performance
- Build a foundation for future research

## Machine Learning Algorithms

- Logistic Regression
- Support Vector Machine (SVM)
- Naïve Bayes
- Decision Tree
- Random Forest

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- NLTK
- TF-IDF Vectorizer

## Dataset

Fake Job Postings Dataset
